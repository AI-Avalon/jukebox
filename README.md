## Install

```shell script
npm install
# or
yarn
```

## Usage

Rename `.env.schema` to `.env` and fill out the values:

```dotenv
DISCORD_TOKEN=
YT_API_KEY=
```

Based on: [discord-music-bot](https://github.com/iCrawl/discord-music-bot)
